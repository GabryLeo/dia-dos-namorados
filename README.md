# 💕 Tinhamu — Página do Nosso Amor

Uma página especial feita com muito amor, dedicada à pessoa mais importante da minha vida. 💖

---

## 📁 Estrutura de Arquivos

Todos os arquivos devem estar na **mesma pasta**:

```
📁 pasta/
  ├── index.html
  ├── die-with-a-smile.mp3
  ├── Imagem do WhatsApp de 2025-06-10 à(s) 20.21.40_09e80d33.jpg
  ├── Imagem do WhatsApp de 2025-06-10 à(s) 20.21.40_1a99822a.jpg
  ├── Imagem do WhatsApp de 2025-06-10 à(s) 20.21.40_1cb63be9.jpg
  └── README.md
```

---

## ✨ Funcionalidades

- 💞 **Contador ao vivo** — mostra há quantos anos, meses, dias, horas, minutos e segundos estamos juntos
- 🖼️ **Carrossel de fotos** — passa as fotos com as setas ← →
- 💌 **Emojis caindo** — animação de corações caindo pela tela
- 🎵 **Música de fundo** — toca automaticamente ao abrir a página
- 💙 **Design especial** — tema azul escuro com brilho

---

## 🚀 Como Abrir

### Opção 1 — Duplo clique
```
Clique duas vezes no arquivo index.html
```

### Opção 2 — PowerShell
```powershell
Start-Process ".\index.html"
```

### Opção 3 — Arrastar
```
Arraste o arquivo index.html para dentro do navegador
```
